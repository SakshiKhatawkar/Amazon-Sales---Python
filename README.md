# 🛒 Amazon Sales Analysis – Python EDA Project

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on an Amazon sales dataset to uncover sales trends, customer behavior patterns, and product performance insights. The objective is to analyze revenue distribution, category performance, and key business metrics using Python.

---

## 🎯 Objective

To explore and analyze Amazon sales data to identify:

* Top-performing **products and categories**
* Sales trends over time
* Revenue distribution patterns
* Customer purchasing behavior
* Factors affecting overall sales performance

---

## 🛠️ Tools & Libraries Used

* **Python**
* **Pandas** – Data cleaning and manipulation
* **NumPy** – Numerical operations
* **Matplotlib & Seaborn** – Data visualization
* **Jupyter Notebook** – Analysis environment

---

## 🔎 Key Analysis Performed

* Data Cleaning & Handling Missing Values
* Descriptive Statistical Analysis
* Revenue & Profit Analysis
* Category-wise Sales Performance
* Monthly / Time-based Sales Trends
* Correlation Analysis between key variables
* Visualization of business insights through charts and graphs

---

## 📊 Key Outcomes

The analysis provides insights into high-revenue categories, seasonal sales patterns, and overall sales distribution. It supports **data-driven business decisions** related to inventory management, marketing strategies, and revenue optimization.
